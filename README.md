## 무엇을 만들고 있나요?

실무와는 달리 개인적으로 조금씩 하는 코딩은 자기 자신이 도메인 전문가가 되어야 한다는 특성을 가지고 있어요.
따라서 지나치게 복잡하거나 어려운 도메인은 다루기 힘들 거라고 생각했어요. 그래서 나름 괜찮을 것 같다고 생각한 도메인이
"개발자를 위한 글쓰기 서비스" 예요.
